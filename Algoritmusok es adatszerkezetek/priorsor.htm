Elsőbbségi sor

  1. Valósítsuk meg a prioritásos sor adattípust 2 verem segítségével. Írjuk meg a műveletek algoritmusát a verem műveleteinek felhasználásával.
  2. Valósítsuk meg a prioritásos sor adattípust 2 sor segítségével. Írjuk meg a műveletek algoritmusát a sor műveleteinek felhasználásával.
  3. (M.o.) Sürgősségi betegellátás. Adott egy input szekvenciális fájlban az ellátásra váró betegek adatai érkezésük sorrendjében. Az adatok alakú párokból állnak, ahol A: betegazonosítók halmaza P:={0,1,…,k} a prioritások halmaza, amely az ellátás sürgősségét szimbolizálja (nagyobb prioritási érték sürgősebb ellátást igényel). Állítsunk elő egy szekvenciális output file-t, amely a fenti párokból áll a betegek ellátásának sorrendjében a következő képen:
    a sürgős eseteket     előre véve a prioritásuk sorrendjében, a kevésbé sürgős esetek követik érkezésük sorrendjében. Az azonos prioritású sürgős esetek sorrendje nem specifikált.
  4. (M.o.) Sürgősségi betegellátás. Adott egy legfeljebb n hosszú input szekvenciális fájlban az ellátásra váró betegek adatai érkezésük sorrendjében. Az adatok alakú párokból állnak, ahol A: betegazonosítók halmaza P:={0,1,…,k} a prioritások halmaza, amely az ellátás sürgősségét szimbolizálja (nagyobb prioritási érték sürgősebb ellátást igényel). Állítsunk elő egy szekvenciális output file-t, amely a fenti párokból áll a betegek ellátásának sorrendjében, a sürgősebb estek a kevésbé sürgős eseteket megelőzve. Az azonos prioritású betegek közül azt vegyük előbbre aki hamarabb érkezett (az input file-ban előrébb volt). A megoldáshoz használjunk prioritásos sort.
