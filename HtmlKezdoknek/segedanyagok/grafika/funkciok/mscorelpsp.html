A KÉP ADOTT SZÍNÉNEK ÁTLÁTSZÓVÁ TÉTELE

 

  1. Töltsük be a képet

  2. Válasszuk ki az ikonnal a megfelelõ színt

  3. A Color similarity (szín hasonlóság) csúszka segítségével azt is beállíthatjuk, hogy ne csak egy bizonyos szín, hanem a hozzá "hasonló" színek is átlátszóak legyenek. A hasonlóságot 0 és 10 % között állíthatjuk be.

  4. Nyomjuk meg az OK billentyût

  1. Töltsük be a képet

  2. Válasszuk ki az ikonnal a megfelelõ színt, és nyomjuk meg az egér jobb gombját

  3. Válasszuk ki a Colors/Set palette transparency/Set transparency value to the current background color menüpontot

  4. Nyomjuk meg az OK billentyût

  1. Töltsük be a képet

  2. Mentsük el GIF formátumban
    File/Save as... , ahol a Save as type mezõben válasszuk ki a Compuserve bitmap (GIF) filetípust

  3. Nyomjuk meg az SAVE billentyût

  4. A Transparency részben válasszuk ki az Image Color opciót

  5. A pipetta segítségével válasszuk ki azt a színt, amit átlátszóvá szeretnénk tenni.

  6. Nyomjuk meg az OK billentyût

Vissza az elejére
