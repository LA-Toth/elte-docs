JAVA alkalmazások 
Anfy Java

 Az Anfy 1.4 egy 40 appletet tartalmazó program. Két részbõl áll: az egyik maga az Anfy Wizard, amellyel "egyszerûen" beállíthatod a paramétereket, majd az elkészített forráskódot csak be kell másolnod a HTML oldalba. A másik módszer Anfy applet készítésére, hogy a korábban bemásolt kódot "manuálisan" átírod, a paramétereket megváltoztatjuk.(Leírást az ASM honlapon találtok, illetve magán az Anfy honlapján ahonnan letöltheted a programot.)

ColorCenter (Forrás: http://www.hidaho.com/colorcenter/cc.html) 
Segítségével megnézheted, hogy egy adott színû háttér mellett milyen színûre érdemes beállítani a szöveg, link, látogatott link, aktív link színeit. Ha sikerült a Neked tetszõ színkombinációt kiválasztani, akkor a <body ...> sort illeszd be a saját HTML oldaladba.

Színkonstansok a HTML-ben  
Kiválaszthatjuk a háttér, a szöveg, és a linkek színét a nekünk tetszõ kombinációban, és megkapjuk, hogy milyen kódot kell beírnunk a forrásba.
